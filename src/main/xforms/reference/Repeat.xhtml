The Repeat Element
Description

The XForms specification allows the definition of repeating structures.

XForms 1.1 Links
Recommendation
Quick Reference
Full Repeat Control (default)

A repeat with no appearance, 'minimal' appearance or 'full' appearance will be rendered vertically with labels on the left and controls on the right.

XForms Markup

<!--XForms UI-->
<xf:repeat id="repeat1" nodeset="repeat/items" appearance="full">
    <xf:input ref="item[1]">
        <xf:label>String:</xf:label>
    </xf:input>
    <br></br>
    <xf:input ref="item[2]">
        <xf:label>Date:</xf:label>
    </xf:input>
    <br></br>
    <xf:input ref="item[3]">
        <xf:label>Boolean:</xf:label>
    </xf:input>
</xf:repeat>
