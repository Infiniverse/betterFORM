Minimal Group

A group with 'minimal' appearance simply wraps the contents of the group in a span Element in the rendered output and is layout-agnostic. It should be used when no layout is implied by the group but only a XForms binding scope should be established.

Vertical Group (rendered as HTML Table)

A group with the custom appearance 'bf:verticalTable' is largely the same as a full group except that is uses tables for left/right aligning labels and controls. This in some cases makes it easier to nest groups as well as optimize the space the respective labels take. With CSS you need to specify a fixed width for the labels which is problematic when labels differ a lot in length. Further labels that do not fit the specified width may corrupt your layout. The use of tables avoids those problems.
