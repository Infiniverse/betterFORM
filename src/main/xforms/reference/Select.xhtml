The select Control
Description
This form control allows the user to make multiple selections from a set of choices. All select controls on this page are bound to the same instance. Thus if you change the value of one control the others will update acccordngly.

Standard Select Control

A select with attribute 'selection="open"' allows the user to add an entry that is not contained in the predefined list of choices. It will be rendered as a select along with an input control. The selection attribute takes precedence over the appearance attribute.
