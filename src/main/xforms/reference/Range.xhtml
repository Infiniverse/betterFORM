Quick Reference
Standard Range Control

Standard rendering for a XForms range control. Fully supports keyboard navigation and mouse wheel scrolling when focussed.

XForms Markup

<!--XForms UI-->
<xf:range ref="item/value" start="1" step="1" end="5" incremental="true">
    <xf:label>a slider</xf:label>
</xf:range>
