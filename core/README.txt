+-------------------------------------------------------------------------+
Apache Ant Targets
+-------------------------------------------------------------------------+

clean
    cleans up core target directory

compile
    compiles all core sources to core/target/classes

package
    creates betterform-core-[version].jar in core/target

test
    execute all JUnit tests

distribute
    create binary and sources distributions

doc
    generates Java API to core/target/doc
