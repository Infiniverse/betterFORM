betterFORM Documentation

The betterFORM documentation app provides several sample, reference and demo forms as well as links to the most important resources regarding betterFORM / XForms.

Feature Explorer

The FeatureExplorer is the betterFORM live documentation. It is itself a XForms that offers a navigation menu with links to many reference XForms that show working examples along with the relevant links to the Spec. A live CSS reference helps with styling forms by showing all available CSS classes. FeatureExplorer will be enhanced continuously as the feature-set grows.
