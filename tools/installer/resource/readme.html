Installer Readme



Quick install

For a quick install choose the components betterFORM XForms Toolkit and follow the install instructions.


After the installation finishes step into the install directory and execute betterFORM.jar



betterFORM XForms Toolkit

betterFORM XForms is a XForms engine for advanced, dynamic forms that run in every major browser without any plugins. You can choose between a standalone version (bundled with jetty) or deploy it to a web container of your choice.


The "standalone" version provides it own unique starter: after the installation switch to the chosen install directory and execute betterFORM.jar to start it


Forms

Installs the betterFORM demo, reference and test forms.
