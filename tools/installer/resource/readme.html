Installer Readme

betterFORM provides several components for installation. Only choosen components will be downloaded and installed.



Quickinstall

For a quick install choose the components jetty and Forms and follow the install instructions. After installation finishs step into the install directory and execute betterform.jar



betterFORM

The betterFORM project. If no other component is selected, betterFORM is installed as web application archive (WAR file) at the choose directory.



Jetty

The Jetty component installs an embeded Apache Jetty server betterFORM runs in. After installation switch to the choosen install directory and execute betterform.jar to start the server



Developer Pack

adds all JavaScript sources uncompiled for full debug capabilities. betterFORM recommends Firebug as client side debugging tool



Forms

This installs the betterForm's demo, reference and test forms. To browse and run the forms point your browser to* http://localhost:45123/betterform/jsp/forms.jsp



Testsuite1.1

The W3C XForms Conformance Test Suite. Browse and execute the tests by pointing your browser to* http://localhost:45123/betterform/jsp/forms.jsp?forms/XFormsTestSuite1.1E1/Edition1



* assuming you run betterFORM within the embedded Jetty Server. If you deploy betterFORM into your own Servlet Container you have to adjust the URLs to the server and port betterFORM runs in
