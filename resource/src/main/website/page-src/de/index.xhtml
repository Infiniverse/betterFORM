Das XForms Toolkit bringt moderne eForms in Ihre Anwendung.

Die betterFORM XML Suite ist eine all-in-one Lösung für komplexe, XML basierte Webanwendungen.

betterFORM is Open Source - keine Lizenzkosten, keine Einschränkungen - eben Open Standard

betterFORM is kompetent in Browser basierten Anwendungen, XML and Webanwedungen. Wir begleiten Sie von der Idee bis zur fertigen Lösung.

Produktivität erhöhen, Kosten senken. Sich auf das konzentrieren 'was zu tun ist' und nicht 'wie es getan wird'. Kein scripting, keine Plugin, läuft in allen gängigen Browsern genauso wie auf Smartphones, Tablets und anderen Geräten.
