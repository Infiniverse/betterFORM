Rich Internet Applications

BetterFORM entwickelt für ein breites Feld von Anwendungsgebieten hochdynamische Rich Internet Applications die in allen gängigen Browsern lauffähig sind. Benutzten Sie ihre Applikation auf einem PC, Laptop, mobilen Endgerät, Kindle, ihrem Fernseher oder wo auch immer Sie wollen. Das abstrakte XForms User Interface ermöglicht betterFORM einfach jede Zielumgebung zu erreichen, von der High-End Web 2.0 Applikation bis hin zu dynamischen Formularen gerendert als PDF, Word oder Excel Dokument.

eForms / XForms

Erstklassige formularbasierte Anwendungen zu entwickeln stellt ein Eckpfeiler unserer Expertise dar. Wir haben für internationale Kunden aus unterschiedlichsten Geschäftsbereichen Formularanwendungen realisiert.

Wie Sie möglicherweise schon selbst festgestellt haben "Formulare sind überall", unsere Lösungen ermöglichen Ihnen eine nahtlose und einfache Integration einer ausgereiften Formularverarbeitung in Ihre bestehende Infrastruktur. Andere Frameworks versagen oft bei hoch-dynamische Benutzeroberflächen wenn es darum geht starke Validierung, Berechnungs-Funktionen und Typisierung zu vereinen oder benötigen hierfür repetitives Skripting, mit XForms erhalten Sie all dies und können zudem das volle Potential von XML Schematas, XPath, XML Events und einer machtvollen, einfach erweiterbaren und deskriptiven Sprache ausschöpfen.

Im Gegensatz zu proprietärer Software basiert das betterFORM XForms Toolkit auf der W3C XForms recommendation und garantiert damit einen langen Lebenszyklus und Interoperabilität mit bestehenden und zukünftigen W3C Standards.

XML Applikationen

Falls das Verarbeiten großer Mengen XML-strukturierter Daten zur Ihren täglichen Aufgaben gehört sind XML-Applikationen ein naheliegender Schritt. "XML auf der ganzen Linie" gibt Ihnen den Vorteil das es nicht nötig ist Daten in Tabellen zu zerlegen, sondern ein simples und elegantes Daten Format kann durch alle Teile ihrer Software Lösung verwendet werden.

Mit XQuery existiert eine leicht erlernbare und benutzbare Abfragesprache, die auf W3C Standards wie zum Beispiel XPath beruht. Durch die Verwendung von weiterer XML-Technologien wie XSLT und XML-FO erweitern sich die Möglichkeiten noch um ein vielfaches.

Haben Sie immer davon geträumt Ihre XML-strukturierten Daten ohne den zusätzlichen Aufwand eines Mappings in einer Datenbank zu hinterlegen? EAI / SOA / EDI

Nicht nur in hoch entwickelten Technologie Feldern wie EAI, SOA oder EDI ist XML aufgrund der leichten Erstellung und Verarbeitung die Lingua franca des Datenaustauschs. Es stellt den gemeinsamen Nenner heterogene System dar auf dem diese ohne die Verwendung proprietärer Schnittellen miteinander gekoppelt werden können. XML erlaubt das aggregieren und austauschen von Information zwischen diesen Systemen.

In Service-orientierte Architekturen werden üblicherweise komplexe interne Logiken hinter einem Service-Ebene versteckt um dann als äußere Fassade für Systemzugriffe zu dienen. SOAP Nachrichten spielen oft eine grundlegende Rolle in diesen Bereichen und werden daher nativ von XForms unterstützt. Dies macht Sie zum idealen Werkzeug um dynamische, validierende Browser Frontends zu erzeugen die direkt mit Ihren Webservice agieren können. Dadurch das ihre Formulare immer valide Daten senden sind zudem zusätzliche Validierungen unnötig.

XForms Applikationen ermöglichen es auf beliebig viele Dateninstanzen aus denn unterschiedlichsten Quellen zu zugreifen und diese als individuelle Ansichten für den Nutzer aufzubereiten, aufgrund dieser Daten zu agieren und sie an weitere Dienste zur Speicherung oder Weiterverarbeitung zu senden. XForms und andere offenen XML Technologien stellen damit eine hervoragende Möglichkeit dar die Funktionalität existierender System um eine ausgereifte Formularverarbeitung erweitern.

Komplette XML Applikationen können sogar noch anspruchsvollerer Bedürfnisse erfüllen. Datenerfassung, Datenverarbeitung, Metadaten Management und Messaging sind nur einige Beispiele der vielfältigen Einsatzmöglichkeiten von XML Technologie.
