Rich Internet Applications

We build plugin-free, highly dynamic Web 2.0 Rich Internet Application based on Open Standards. These will run in any major Browser on the market. This means that one Application has to be developed only once and can than be run everywhere where a modern browser is available. So you can instantly reach your Users on mobile Devices, in their Living Room and their Office not needing them to install any client software.

At betterFORM we build highly dynamic Rich Internet Applications for any use case running in every common browser. Enjoy your application on a Desktop PC, Laptop, Mobile Device, your home TV or wherever you want it. Due to the XForms User Interface is abstract betterFORM can easily target any end device, from the high end web 2.0 application to dynamic form rendered as PDF, Word or Excel file.

eForms / XForms

Building vibrant form-based Applications is one of the cornerstones of our expertise, we use and used forms in projects for customers from lines of business like eHealth, eGovernment and eCommerce. As you may have experienced for yourself "Forms are everywhere". Our Solutions provided a plugable and seamless integration into your current setup up(?). They give you the advantage of sophisticated validation and calculation mechanisms, easy access to databases and backend systems with the possibility to present collected or already existing data inside a browser or even as normal documents like PDF, Word or Excel files.

betterFORM eForms provides a generic standardized XForms based solution to create dynamic forms for any kind of user input within Web-Applications. No custom scripting is needed to validate user inputs, calculate data or to create fancy Web 2.0 ui controls. Thanks to strong typing controls are rendered depending on how they are typed (e.q. data typed as boolean is rendered as a checkbox)

Unlike proprietary software betterFORM eForms is based on the open W3C XForms standard guaranteeing a long life time and interoperability with existing and future XForms implementations.

XML Applications

If processing large amounts of data in XML-structures is your livelihood utilizing XML-Application is a natural way to go. "XML from front to back" will give one the advantage that there is no need to shred data into tables rows, just a single elegant data format trough all parts of a software solutions. With XQuery there exists an easy to use and learn query language based on established W3C-standards like XPath. Appliance of other XML-technologies like XSLT and XML-FO will broaden the capabilities even more.

Data Management
