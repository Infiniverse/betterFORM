Downloads

Der empfohlene Weg um betterFORM zu installiern ist die Verwendung des WebInstallers. Er führt Sie durch die Installation und bietet ihnen die Möglichkeit nur ausgewählte Komponenten zu installieren.

Klicken Sie den Knopf, um den WebInstaller zu laden.

Installations Möglichkeiten

  • Als Web Archive (.war) einsetzbar in jedem beliebigen Web Container
  • gebündelt mit Jetty als eine selbständige Anwendung
  • gebündelt mit Jetty und der eXist XML Datenbank

System Voraussetzungen

  • Sie benötigen Java in der Version 1.5 oder höher um betterFORM auszuführen.

Ausführen von betterFORM

  1. Wechseln Sie in das Verzeichnis in das Sie den Installer geladen haben
  2. Starten sie den Installer durch einen Doppelklick auf das betterFORM-install.jar
  3. Folgen Sie den Anweisungen des Installers, um betterFORM zu installieren
  4. Wechseln Sie in das Installationsverzeichnis
  5. Doppelklicken Sie anschließend auf das betterFORM.jar um den Server zu starten
  6. Mit einem Doppelklick auf das betterFORM Logo starten Sie ihren Browser

Das Benutzerhandbuch ist eine Pflichtlektüre für jeden neuen Nutzer. Sein Fokus liegt im speziellen auf den Themen: Erstellung von XForms, sowie Gestaltung und Seitenlayout. Er sollte ihnen einen guten Überblick verschaffen, welche Funktionalitäten Ihnen die Platform bietet.

Als Hilfe zur Seitengestaltung und -layout ist die CSS Referenz gedacht. Sie beinhaltet alle CSS-Klassen die dem Benutzer zum Verändern des Erscheinungsbild und der Benutzeroberfläche zur Verfügung stehen.

Das Entwickler Handbuch ist für Architekten und Entwickler gedacht, die Informationen über die betterFORM Architektur, das Erzeugen des Programms aus dem Quellcode, die verschiedenen Möglichkeiten der Erweiterung der Grundfunktionalität, als auch die Benutzung der Java API suchen.
