Product

Features

  • easy installation
  • dynamically loaded subforms
  • rapid webapplication development
  • absolutely no script programming required
  • fully declarative programming model
  • no plugins - runs in every major browser withhout installation
  • advanced datatyping, validation and calculation support
  • dependency engine for complex validations
  • complient to XForms 1.1 Recommendation
  • seemless integration
  • deployable in a separate webcontext
  • fully accessible UI controls
  • internationalisation and Localisation support
  • Open Source under business-friendly BSD or Apache 2 license
  • professional support available

Sourceforge

The betterFORM project is hosted at Sourceforge and uses its mailing-list and download features.

The users list answers quesions regarding the usage of the betterFORM toolkit, its installation and problems that arise during operation as well as integration questions. To subscribe please visit this URL:

https://lists.sourceforge.net/lists/listinfo/betterform-users

The developer list is dedicated to discussions about the architecture, new features and development internals. It is NOT intended for developers that use the toolkit but for developers that like to extend and improve betterFORM itself. To subscribe to this list please visit:

https://lists.sourceforge.net/lists/listinfo/betterform-developer
