System Requirements

  • Es wird Java 1.5 oder höher benötigt, um betterFORM auszuführen.

betterFORM starten

  1. Wechseln Sie in das Installtions-Verzeichnis
  2. Mit einem Doppelklick auf 'betterform.jar' führen Sie betterFORM aus
  3. Es erscheint ein kleines Anwendungsfenster mit dem betterFORM Logo.
  4. Klicken Sie auf das Logo, um Ihren Browser mit der Adresse 'localhost:8080'zu starten.

Erste Schritte

Um sich mit betterFORM vertraut zu machen, empfehlen wir den XFormsFeatureExplorer. Dieser zeigt live-Beispiele aller Controls und Container mit Markup-Beispielen und Beschreibungen. Der Feature Explorer wird ständig um neue Beispiele erweitert, um neue betterFORM Funktionen zu demonstrieren.

Der User Guide sollte von jedem Anwender gelesen werden und wendet sich insbesondere an XForms Autoren, die die Grundlagen von betterFORM kennenlernen möchten.

Der Developer Guide richtet sich an Architekten und Enwickler, die Informationen über die Architektur von betterFORM suchen. Hier wird beschrieben, wie betterFORM compiliert wird, wie und wo Erweiterungen gemacht werden können und wie die API genutzt werden kann, um komplexe Anforderungen zu bedienen.
