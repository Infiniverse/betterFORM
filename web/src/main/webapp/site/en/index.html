betterFORM is an platform for the rapid development of web applications completely based on open standards like XML, XForms 1.1, XPath 2, XSLT 2, XQuery and others. It is itself licensed as open source software.

The declarative approach avoids heavy use of scripting and reduces maintainance cost. Attractive, consistent and accessible user interfaces combined with a model-driven architecture address even complex project requirements.

betterFORM XRX is a complete XML processing server combining most of the W3C XML standards in an open, extensible platform. Applications that use XML from front to back are easy to maintain and flexible to change.
