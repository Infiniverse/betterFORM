7. Juni 2010

Das betterFORM Projekt hat betterFORM 3.1 lighSteelBlue XRX Edition veröffentlicht. Mit dem neuen Installer steht ihnen ein Komplettpaket aus eXist XML Datenbank und betterFORM zur Verfügung.

Weitere Informationen zu diesem Thema finden Sie hier.

betterFORM erlaubt die schnelle, deklarative Entwicklung von dynamischen Web Anwendungen aufbauend auf dem W3C XForms Standard.

betterFORM ist als Open Source unter der BSD Lizenz verfügbar
