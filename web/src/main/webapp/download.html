Installation Options

  • Web Archive (.war) deployabel in any webcontainer
  • bundled with Jetty as a standalone application
  • bundled with Jetty and eXist XML Database

System Requirements

  • You need Java 1.5 or better to run betterFORM

Run betterFORM

  1. Switch to your installation directory
  2. Execute the betterform-install.jar by double-clicking it
  3. Point your browser to localhost:8080/betterform

Getting started

To get started we recommend to have a look at the XFormsFeatureExplorer which shows live examples of all available controls and containers along with markup and descriptions. The FeatureExplorer will constantly be extended and updated as betterFORM moves on.

The User Guide is a must-read for the every new user and especially focuses on XForms authoring, styling and page layout. It should give a overview of what the platform offers out of the box.

The CSS Reference is meant as a helper when styling and layouting pages. It lists all CSS classes that are available for customizing the look and feel of the user interface.

The Developer Guide is for architects and developers seeking information about the architecture of betterFORM, building from sourcecode, the various ways to extend the functionality and usage of the Java API.
