betterFORM lightSteelBlue (Version 3.0)

Requirements

  • You need a Java (Version 1.5 or higher) installation to execute betterFORM and the WebInstaller.
  • Make sure your system is configured to execute jar-files to use the WebInstaller

WebInstaller

Download just what you need. The WebInstaller is itself a small download which lets you choose which components to install.

It will guide you through the installation process and supports a variety of installation options including Jetty, the eXist XML Database, the Dojo Toolkit and more.

For a quick installation select 'betterFORM', 'Jetty' and 'Forms' at the "Select Installation Packages" screen and finish the installation. Switch to the choosen installation directory, execute the start.jar and point your browser to localhost:8080/betterform. That's it!

Web Archive (war-file)

For those that already have a webcontainer this download provides an optimized war build ready for deployment.

The Core (jar-file)

Use the betterFORM core to process XForms within your Java application.
