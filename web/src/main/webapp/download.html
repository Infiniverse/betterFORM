Install

You can install betterFORM as standalone application (Jetty), Web Application Archive to be executed within any standard servlet container or bundled with the eXist XML Database.

The installer will guide you through the installation process and only download selected components.

For a quick install select the components 'betterFORM', 'Jetty' and 'Forms' at the "Select Installation Packages" screen and finish the installation.

Switch to the chosen installation directory, execute the start.jar and point your browser to localhost:8080/betterform. That's it!

To start working with betterFORM we recommend to take a look at the betterFORM User Guide

Requirements

You need Java to run betterFORM
Make sure your system is configured to execute jar-files to use the WebInstaller
Web Application Archive

For those that already have a webcontainer betterFORM provides an optimized web application archive (WAR) build ready for deployment.

download war
 
XForms Core

for XForms enabled Java applications simply download the betterFORM Core and at it on your classpath.
