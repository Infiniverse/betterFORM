Quick Install

You need theJava Runtime Environment (JRE) 5 or higher to run betterFORM.

on your marks!

Download betterFORM Version 3.0 If you are running an earlier version of betterFORM - or any other application running on port 8080, you need to shut it down.

get set!

Double-click on the downloaded betterform-installer.jar to execute it. If a warning dialog opens, click Continue and follow the install instructions.At the "Select Installation Packages" screen select 'betterFORM', 'Jetty' and 'Forms' and finish the installation.

go!

Switch to the choosen installation directory, execute the start.jar and point your browser to localhost:8080/betterform. That's it!


If you experience any problems make sure start.jar is executable, the Java Runtime Environment is installed and take a look at chapter "Troubleshooting" of the betterFORM User Guide
