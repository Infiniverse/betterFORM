Die betterFORM 3.1 lighSteelBlue XRX edition wurde veröffentlicht. Mit dem neue Installer steht ihnen ein "ready-to-go" Komplettpaket aus der eXist XML Datenbank und betterFORM zur Verfügung.

Eine neue Demo Anwendung zeigt Ihnen wie Sie schnell und effizent Web-Anwendungen mit der neuen XForms Rest XQuery Architektur entwickeln können.

Weiter Informationen zu diesem Thema finden Sie hier.

betterFORM erlaubt das einfache Erzeugen von hoch-dynamischen Web 2.0 Benutzeroberflächen mit ansprechenden Bedienelementen und Layout. Damit haben Sie die Möglichkeit deklarativ vollständige Webapplikationen zu erstellen die Ihnen Validierung, Kalkulation, Aktionen und Ereignissen bieten.
