Produkt

Eigenschaften

  • einfache Installation
  • dynamisches laden von Subformularen
  • schnelle Entwicklung Webanwendungen
  • keine Skriptprogrammierung nötig
  • ein voll deklaratives Programmiermodel
  • keine plugins - läuft ohne Installation auf allen gänigen Browsern
  • erweiterte/fortschritliche Unterstüzung für Typisierung, Validierung und Berechnungen
  • dependency engine für komplexe Validierungen
  • erfüllt die XForms 1.1 recommendation
  • nahtlosse Integration
  • in einem seperaten webcontext einsetztbar
  • fully accessible UI controls
  • Ünterstüzung für Internationalisierung und Lokalisierung
  • Open-Source unter der unternehmensfreundlichen BSD oder Apache 2 Lizenz
  • Professioneller Support verfügbar

Zugang für Entwickler

Der betterFORM Quellcode kann über unser Subversion Repository bezogen werden. Die Adresse hierfür lautet:

https://www.betterform.de/svn/betterform

Installieren Sie Subversion und führen Sie folgenden Befehl in einem Terminal aus um die aktuellen Quellen des betterFORM Projekts zu beziehen.. svn checkout https://betterform.de/svn/betterform/trunk Sie können auch einen Subversion Klienten ihrer Wahl verwenden.


Alle betterFORM Quellen und deren Revisionen können Sie über den folgenden Link betrachten:

https://www.betterform.de/trac/browser

Die Weiterentwicklung der betterFORM Quellen findet im 'trunk' statt. Die einzelnen veröffentlichten Versionen sind über die mit "R-" beginnenden Ordnern im 'tags' Verzeichnis ereichbar, z.B. 'R-3-1-0' für die aktuelle Version lightSteelBlue.

Falls Sie bereit sind sich an der Weiterentwicklung des betterFORM Projekts zu beteiligen, kontaktieren Sie uns über die Mailing Liste.

Mailing Lists

Das betterFORM Projekt wird von Sourceforge gehostet und benutzt deren E-mail Verteiler und Download Angebote.

Die Benutzer Liste beantwortet Fragen im Zusammenhang mit der Benutztung des betterFORM Toolkits, seiner installation, Probleme die im laufenden Betrieb auftretten und Fragen zur Integration dessen. Um dem E-mail Verteiler beizutretten besuchen Sie bitte die folgende URL:

https://lists.sourceforge.net/lists/listinfo/betterform-users

Die Entwickler Liste ist für die diskusion über die Architektur, neue Features und Entwicklungungsinterna reserviert. Sie ist NICHT für Entwickler gedacht die das Toolkit verwenden, sondern für Entwickler die sich an der weiterentwicklung des betterFORM Projekts beteiligen wollen. Um dem E-mail Verteiler beizutretten besuchen Sie bitte die folgende URL:

https://lists.sourceforge.net/lists/listinfo/betterform-developer
