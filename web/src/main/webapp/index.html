the XForms way to build the web
5th March 2010
The betterFORM project released Version 3.0 codenamed 'lightSteelBlue' to the public. It combines a full XForms 1.1 implementation with the Dojo toolkit for enterprise-level solutions.
betterFORM allows easy creation of highly dynamic Web 2.0 user interfaces with attractive controls and layout. You can add validations, calculations, actions and events to build complete webapplications in a declarative way.
